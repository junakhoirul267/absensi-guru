📖 Panduan & Catatan

📌 Cara Penggunaan:

  1. Buka tab Absensi, pilih tanggal, cari nama guru, lalu klik nama guru.
  2. Isi status untuk setiap jam (Hadir / Izin / Sakit / Alpha).
  3. Klik Simpan Absensi — data tersimpan di browser (LocalStorage).
  4. Untuk melihat rekap bulanan, buka tab Rekap Bulanan.

☁️ Data Online:

Aplikasi ini menggunakan GitHub database.json sebagai database cloud (gratis, tanpa perlu daftar akun). Semua perubahan data langsung terlihat oleh semua pengguna. Data juga tetap aman di LocalStorage browser sebagai cadangan.
